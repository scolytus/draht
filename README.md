
A wire loop game for the Raspberry Pi.

Details will follow.

Quick Install
=============

1. login to your pi
1. load the install script: `wget https://raw.githubusercontent.com/scolytus/draht/master/install.sh`
1. make it executable: `chmod a+x install.sh`
1. run `./install.sh`
1. reboot your pi
1. have fun :)

Require Packages
----------------

1. python3
1. gatling
1. alsa-utils
